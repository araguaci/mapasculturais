# Componente `<entity-card>`
Lista informações sobre uma entidade no card

## Propriedades
- *Entity **entity*** - Entidade

### Importando componente
```PHP
<?php 
$this->import('entity-card');
?>
```
### Exemplos de uso
```HTML
<!-- utilizaçao básica -->
<entity-card :entity="entity"></entity-card>

```
