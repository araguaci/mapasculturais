# Componente `<mc-status>`
Componente para exibição do status com sua respectiva cor
  
## Propriedades
- *String **statusName*** - Nome do status

### Importando componente
```PHP
<?php 
$this->import('mc-status');
?>
```
### Exemplos de uso
```HTML
<!-- utilizaçao básica -->
<mc-status status-name="pendente"></mc-status>

```
